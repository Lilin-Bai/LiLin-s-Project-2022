# Arduboy
Arduboy Gameplayer(ATMEga32U4) with Flash

##########

项目概述


项目根据海外开源项目加以改进，添加128M的Flash，相较于原版只能存1个游戏显著提升，现可烧录300+游戏，由于ATMega32U4的强悍性能，可以畅玩很多游戏，体验极佳。


##########

制作与调试
